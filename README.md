## 🚀 Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

```bash
git clone https://github.com/CEO-LAYSON/layson-portifolio.git
cd layson-portifolio
npm install
npm run dev
```

Open your browser at `http://localhost:5173`
